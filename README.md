# Gemstone Jewellery Website Conversion Guide

## Project Overview

Convert the existing **LapCity E-Commerce Template** into a **Luxury Gemstone Jewellery Website**.

> **IMPORTANT:** This is a **content and branding conversion**, **NOT** a redesign.

The template architecture has already been professionally designed and must remain intact.

---

# Primary Objective

The entire UI/UX should feel like a premium luxury jewellery brand while preserving the existing layout, responsiveness, interactions, animations, and component architecture.

---

# DO NOT MODIFY

The following **must remain exactly as they are**:

- Overall page layout
- Section order
- Grid systems
- Responsive design
- Card layouts
- Navigation structure
- Footer structure
- Hero slider mechanics
- Hover animations
- Scroll animations
- Button interactions
- Carousels
- Sliders
- JavaScript functionality
- Component spacing
- Mobile experience

The goal is **only** to change branding, colors, images, icons, and content.

---

The current theme is based around a cyberpunk electronics aesthetic.

Replace it with a premium luxury gemstone identity.
